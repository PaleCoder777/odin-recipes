# odin-recipes

Introduction

This project serves as a very basic html website set up.
The theme revolves around food recipes.
It consists of a homepage, which lists 3 recipes:
    Pizza Dough
    Peanut Butter and Jelly Sandwich
    Ham and Cheese Sandwich

The home page (index) links to all the recipes, and the recipes link back to homepage.



Skills Demonstrated
+ Github commands
+ Consistent Commits and Clear Messaging
+ HTML coding (boilerplate, elements, attributes)
+ Project Management (how to structure repository, page structure comments to copy over to new pages)
